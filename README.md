📌 Descripción

Este proyecto consiste en el desarrollo y simulación de una vivienda inteligente capaz de automatizar diferentes funcionalidades del hogar mediante sensores, actuadores y lógica programada.

Durante el desarrollo se implementaron distintos módulos electrónicos y extensiones compatibles con Arduino para recrear comportamientos reales de una Smart Home, incluyendo automatización, monitorización y control de dispositivos.

El objetivo principal fue aplicar conocimientos de programación, electrónica y sistemas embebidos para construir soluciones domóticas funcionales y escalables.
